DESIGN PHILOSOPHY:

Luxury = restraint

Less motion = more premium

---

MOTION TOKENS:

Fast:
0.2s → hover

Normal:
0.4s → transitions

Slow:
0.6s → page entry

Ease:
cubic-bezier(0.22, 1, 0.36, 1)

---

HOVER SYSTEM:

Cards:
transform: scale(1.02);
transition: 0.3s ease;

Buttons:
transform: translateY(-2px);

Glow:
box-shadow: 0 0 20px rgba(255, 42, 42, 0.2);

---

SCROLL ANIMATION CLASS:

.fade-up {
  opacity: 0;
  transform: translateY(20px);
  transition: all 0.6s ease;
}

.fade-up.visible {
  opacity: 1;
  transform: translateY(0);
}

---

IMAGE BEHAVIOR:

- No parallax
- No zoom on scroll
- Only fade-in

---

TYPOGRAPHY MOTION:

- NO animated letters
- NO typing effects
- Only fade-in blocks

---

LAYOUT RULE:

Whitespace > animation

---

USER EXPERIENCE GOAL:

User should feel:
"this is smooth and premium"

NOT:
"this is animated"f
